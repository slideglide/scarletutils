# Scarlet Utils
A mini menu packed with tools for showcasing and botting.

## Usage
The default keybind to open the menu is **F**. This keybind is customizable in both the mod's settings and Geode's keybind settings.

## Features
- Automatic Swift Click Tools
- Advanced Autoclickers
- Maintain Gravity
- Layout Mode
- Other Features
